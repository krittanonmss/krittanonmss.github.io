# Kritanon Museesut - Portfolio Website

This is my personal portfolio website showcasing my skills, projects, and contact information.  
The website is built with HTML, CSS, and JavaScript, and is hosted on GitHub Pages.

## Features

- Introduction section with profile picture and bio
- Skills section categorized by:
  - Programming Languages
  - Tools & Collaboration
  - Frameworks & Technologies
- Projects section with draggable carousel and project cards
- Contact section with email, phone number, and social links (Instagram, LinkedIn, LINE)
- Smooth scrolling navigation
- Responsive design for desktop and mobile

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Icons & Logos:** DevIcons, raw SVG icons
- **Tools:** Git, GitHub, GitHub Pages
